## File Commands

> Delete file `rm file-name`

`$ rm .m2/settings.xml`

> Copy file1 to file2 `cp file1 file2`

`$ cp .m2/settings.xml .m2/settings.xml.ecomm`

> Rename source to dest `mv file1 file2`

`$ mv .m2/settings.xml .m2/settings.xml.ecomm`

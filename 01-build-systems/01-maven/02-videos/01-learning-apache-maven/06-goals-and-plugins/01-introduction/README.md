```
Plugin = class
Goal = method
`mvn plugin:goal` = `mvn clean:clean`
```

> find out plugin and plugin-goals from `super` pom

> super pom - `mvn  help:effective-pom`

get `clean` plugin that has only `clean` goal out of the super pom below

```
      <plugin>
        <artifactId>maven-clean-plugin</artifactId>
        <version>2.5</version>
        <executions>
          <execution>
            <id>default-clean</id>
            <phase>clean</phase>
            <goals>
              <goal>clean</goal>
            </goals>
          </execution>
        </executions>
      </plugin>
```

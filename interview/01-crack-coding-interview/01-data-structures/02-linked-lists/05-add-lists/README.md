# add-lists

maven-archetype-quickstart project

## Build package

> `mvn package` or `mvn clean install`

## Run the application

> `java -cp target/app-1.0-SNAPSHOT.jar com.company.app.App`

```
$ java -cp target/app-1.0-SNAPSHOT.jar com.company.app.App
#addListsReverseOrderTest
   7 1 6 
 + 5 9 2 
   2 1 9 
#addListsTest
   7 1 6 
 + 5 9 7 
 1 3 1 3 
```

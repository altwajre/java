# is-subtree

maven-archetype-quickstart project

## Build package

> `mvn package` or `mvn clean install`

## Run the application

> `java -cp target/app-1.0-SNAPSHOT.jar com.company.app.App`

```
Root: 5
Bfs: 5 2 7 1 3 6 8 4 9 
#subTreeTest
is 2 a subtree of 5? true
#notSubTreeTest
is 18 a subtree of 5? false
```

# hanoi-towers-move-disks

maven-archetype-quickstart project

## Build package

> `mvn package` or `mvn clean install`

## Run the application

> `java -cp target/app-1.0-SNAPSHOT.jar com.company.app.App`

```
$ java -cp target/app-1.0-SNAPSHOT.jar com.company.app.App
--Print tower0 disks--
0
1
2
--done--
--Print tower1 disks--
Empty
--Print tower2 disks--

Move disk 2 from 0 to 2

done
```

testng-project
==============

## Add testng dependency to pom.xml

```
    <dependency>
      <groupId>org.testng</groupId>
      <artifactId>testng</artifactId>
      <version>6.1.1</version>
      <scope>test</scope>
    </dependency>
```

## Run Test

> Right click `TestNGTest` class, and click `Run 'test()'`

# comparable

## Create package

> `mvn package`

## Run

> In intellij, Run, Run 'App'

> `java -cp target/app-1.0-SNAPSHOT.jar com.company.app.App`

```
$ java -cp target/app-1.0-SNAPSHOT.jar com.company.app.App
Employee - Id: 1, Name: Dick, Salary: 80.0
Employee - Id: 0, Name: Tom, Salary: 100.0
Employee - Id: 2, Name: Harry, Salary: 200.0
```
